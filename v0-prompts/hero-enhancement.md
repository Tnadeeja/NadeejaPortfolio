# v0.dev Hero Section Enhancement

## PROMPT
Create a premium hero section for a data science & full-stack developer portfolio.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Dark + light mode

## Design Requirements
- Glassmorphism with backdrop blur
- Animated gradient background
- Floating elements with parallax
- Professional photo frame
- Green accent glow effects

## Content Structure
```tsx
<div className="hero-container">
  <div className="floating-elements">
    {/* Animated geometric shapes */}
  </div>
  <div className="content">
    <div className="profile-image">
      {/* Professional photo with frame */}
    </div>
    <div className="text-content">
      <h1>Thamindu Nadeeja</h1>
      <p>Data Science & Full Stack Developer</p>
      <div className="cta-buttons">
        {/* Primary and secondary CTAs */}
      </div>
    </div>
  </div>
</div>
```

## Animation Requirements
- Parallax scrolling effect
- Floating geometric shapes
- Text reveal on scroll
- Smooth hover transitions
- Glow effects on interaction

## Style Guidelines
- Minimal and professional
- Premium glassmorphism
- Green accent color (#22c55e)
- Responsive design
- Accessibility focused

## Interactive Elements
- Smooth scroll to sections
- Download resume button
- Social media links
- Theme toggle integration
